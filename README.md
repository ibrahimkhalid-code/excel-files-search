This project searches for all Excel and Word files on your PC and displays the found files in a Listbox GUI.
The program scans selected drives (such as C:\ and D:\) and lists all files with the following extensions:

.xls
.xlsx
.doc
.docx

You can open any file directly by clicking on it from the list.

Technologies Used
Python 3.10
Tkinter (GUI)
OS module
Threading
